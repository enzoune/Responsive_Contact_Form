# ✉️ Professional Responsive Contact Form

> Uma interface de contato sofisticada e adaptável, projetada para facilitar a comunicação entre usuários e empresas através de um design limpo e funcional.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **Página de Contato** que combine informações institucionais (endereço, telefone e e-mail) com um formulário de mensagem direto, garantindo que o usuário tenha múltiplos pontos de contato em uma única visualização.

O layout foi planejado para manter a clareza informativa em diferentes resoluções, utilizando uma divisão lógica entre dados de suporte e campos de interação.

## ✨ Destaques e Diferenciais Técnicos

* **Arquitetura de Duas Colunas:** Divisão inteligente entre o `left-side` (informações de contato) e o `right-side` (formulário de mensagem), otimizando o fluxo de leitura do usuário.
* **Uso Semântico de Elementos de Formulário:** Implementação de campos de entrada específicos e área de texto (`textarea`), demonstrando domínio sobre a coleta de dados estruturados.
* **Integração de Iconografia (Font Awesome):** Uso de ícones vetoriais para reforçar a semântica visual de endereço, telefone e e-mail, melhorando a experiência do usuário (UX).
* **Design Centrado no Usuário:** Inclusão de placeholders descritivos e textos de apoio que guiam o visitante durante o preenchimento.
* **Modularidade e Escalabilidade:** Código organizado em blocos de classes como `.input-box` e `.details`, o que facilita a manutenção e a aplicação de estilos consistentes.
* **Organização Profissional:** Separação estrita entre a estrutura de marcação (`index.html`) e a identidade visual (`css/style.css`), seguindo os padrões de engenharia de software.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação e organização de componentes de formulário.
* **CSS3:** Gerenciamento de layout, tipografia e design responsivo.
* **Font Awesome (v5.15.2):** Biblioteca de ícones para suporte visual e semântico.
